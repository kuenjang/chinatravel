# 專案總結：旅居中國指南 (China Sojourn Guide)

本文件摘要了「旅居中國指南」專案的核心資訊，供 AI 助手或開發人員快速了解專案背景與架構。

## 1. 專案目標
打造一個視覺精美、具高級感 (Premium Feel) 的響應式單頁網站 (SPA)，為數位游牧者提供經濟實惠的中國旅居行程資訊與生活指南。

## 2. 技術棧 (Tech Stack)
- **前端框架**：React 19 (使用 Vite 建立)
- **樣式處理**：Vanilla CSS (純 CSS 實作，包含毛玻璃特效與動態美學)
- **開發工具**：Vite, ESLint
- **部署平台**：Netlify (支援自動化部署)
- **版本控制**：Git / GitHub

## 3. 專案結構
- `src/`：原始碼目錄
  - `components/`：可重用 UI 元件
    - `Hero`：具備全屏影像與動態標題的首屏。
    - `DestinationCard`：行程推薦卡片（大理、成都、泉州）。
    - `TipsSection`：條列式實用指南。
  - `App.jsx`：主應用程式入口，整合所有頁面區塊與資料。
  - `index.css`：全域設計系統（顏色變數、字體、毛玻璃特效定義）。
- `public/assets/`：存放專案所需的影像資源（AI 產生的景觀圖）。
- `dist/`：正式環境建置輸出的資料夾。

## 4. 關鍵功能
- **動態視覺效果**：具備 Fade-in 動畫、懸浮縮放、及平滑滾動。
- **響應式佈局**：完美支援行動裝置與電腦端。
- **實用指南整合**：涵蓋支付寶/微信支付、網路漫遊、住宿登記等必備資訊。

## 5. 部署配置
- 專案根目錄包含 `netlify.toml`，預定義了建置指令 (`npm run build`) 與發佈資料夾 (`dist`)。
- `vite.config.js` 已設定 `base: './'` 以提高跨平台部署的相容性。
